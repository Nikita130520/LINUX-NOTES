INPUT-OUTPUT REDIRECTION
--------------------------------

standard input(stdin)
-----------------
the stdin stream is numbered as stdin(0)

standard output
--------------------
stdout(1)

standard error
-----------------
stderr(2)


INPUT REDIRECTION
---------------------

<stdin

ex: cat <filename

    wc  <filename

OUTPUT REDIRECTION
--------------------------
>stdout

ex: echo "hello wrold" >filename


ERROR REDIRECTION
---------------------------
2>stderr
scho "hello world" 2>error
